# Snake_and_Ladder_Group_A
Snake &amp; Ladder Game – A classic board game built using Python and Tkinter. It features a graphical board, real-time player movement, snakes, ladders, and dice rolls with interactive UI elements. Designed for fun gameplay and hands-on learning with GUI programming.
# 🎲 Snake & Ladder Game (Python GUI)

A fun and interactive Snake & Ladder game built using **Python**, **Tkinter**, and **Pygame**.  
This is my **first GUI project** with sounds and animations for snakes, ladders, and dice rolls.

---

## 🧩 Features

- ✅ 2–4 Players supported
- ✅ Custom player names
- ✅ Real board movement with token positioning
- ✅ Sound effects for:
  - 🎲 Dice roll
  - 🐍 Snake bite
  - 🪜 Ladder climb
  - 🏆 Game winner
- ✅ Auto final ranking popup at game end
- ✅ Clean UI with split panel layout

Install dependencies before running the game:

🚀 Game Rules
Rule	Description
🎯 Goal	Reach exactly 100 to win
🐍 Snake	Slide down to a lower position
🪜 Ladder	Climb up to a higher position
❌ Over 100	Move not allowed
